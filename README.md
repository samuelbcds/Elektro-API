# Project: Elektro API

## End-point: Criar User
Cadastra um usuário no sistema preenchendo suas informações (nome, dataNasc, email, senha) no Body da requisição
### Method: POST
>```
>http://localhost:3333/usuarios
>```

## End-point: Atualizar User

>```

>```

|Param|value|
|---|---|
|1|null|

## End-point: Atualizar Pedidos
Atualiza o status de um pedido, dado seu Id no paramêtro da URL e o seu novo status no Body da requisição

>```

>```

|Param|value|
|---|---|
